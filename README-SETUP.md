# Setup Guide

This repository is a self-hosted 5etools deployment with optional ngrok sharing.

## Prerequisites

- Node.js 24+
- Git
- ngrok installed and authenticated

## Install

```bash
npm install
```

## Start Services

### Local app

```bash
npm start
```

### Tunnel (new terminal)

```bash
npm run tunnel
```

## Common URLs

- Local app: http://localhost:3000/
- Local analytics dashboard: http://localhost:3000/analytics
- Initiative tracker: http://localhost:3000/inittrackerplayerview.html?auto=1
- Public app: https://YOUR_NGROK_URL.ngrok-free.dev/

## Optional Windows launchers

- start-server.bat
- start-ngrok-tunnel.bat
- auto-restart-tunnel.ps1

## Troubleshooting

- If tunnel returns 502, ensure app server is running on port 3000.
- If images look broken, run `npm run audit:images` and refresh the page.
- If ngrok warning page appears, use the direct tunnel URL generated by `start-tunnel.js`.

## Updating from upstream

```bash
git fetch upstream
git merge upstream/main
```

Upstream: https://github.com/5etools-mirror-3/5etools-src
